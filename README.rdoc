= Cinatra

Cinatra is a Sinatra like command base app library.

== Usage

  require 'cinatra'

  command 'todos' do |arg|
    show_todos
  end

sub command:

  command 'todo add' do |arg|
    TODOS << arg
  end

== Run

  ruby app.rb

== Installation

  gem install cinatra
